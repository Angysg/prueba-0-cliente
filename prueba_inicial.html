Una pagina web para una asociacion de animales donde te pregunte si quieres adoptar o donar algo. Cuando seleccionas adoptar, te pregunta que buscas (gato o perro) y si no tienes ningun requisito más a buscar. Si la persona no tiene requisitos, que vaya directamente a la base de datos para buscar el animal que lleva mas tiempo allí y mostrarselo a la persona. Si la persona si tiene requisitos, le pedira unos ciertos datos al usuario para buscar en la base de datos los animales que concuerdan son su búsqueda y se lo muestra. Los datos de entrada serian: - Si quieres adoptar o donar. - Adoptar perro o gato. - Si tiene requisitos o no. - Rango de edad del animal, sexo del animal, etc. Los datos de salida serian los animales disponibles para adoptar.
